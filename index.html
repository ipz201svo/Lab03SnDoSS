Сертифікації

Скаковський Владислав

Я вже маю сертифікацію Azure Fundamentals. Була можливість отримати ваучер на екзамен AWS Cloud Practitioner, тому буду його складати вже 7.04 :)

А в подальшому планую отримати сертифікацію Azure Developer Associate і можливо Azure Solutions Architect Expert. Хоча на поточному місці роботи Azure проходить повз мене, але основи роботи із ним будуть корисними у бідь-якому разі. Так як я .NET розробник, то саме Azure як клауд провайдер мені підходить більше за все.

Azure наразі є другим за популярністю клауд провайдером після AWS. Я також трохи пробував GCP, але він мені не подобається. Найбільше сподобався Azure.

Сертифікація Azure Developer Associate показує що розробник має доволі глибокі знання із Azure та знає коли і вміє як застосовувати його сервіси та best-practices. Це дуже корисно для роботодавця, який хоче знати що його робітник знає і вміє.
